# leetcode-hot100-interview-notes

## 仓库用途

本仓库是本人面试算法能力沉淀仓库，整理 LeetCode Hot100 高频面试题的标准化复盘文档，用于巩固算法基础，应对 NLP、大模型算法工程师、LLM 应用工程师相关笔试与面试。

## 目录结构

```text
leetcode-hot100-interview-notes/
├── README.md
├── Day01_两数之和.md
├── Day02_字母异位词分组.md
├── ...
└── Day100_寻找重复数.md
```

文件命名采用 `DayXX_题目名称.md`，便于按刷题顺序复盘，也便于面试前快速定位专题。

## 单篇文档内容规范

每篇题解文档统一包含以下模块：

- 题意分析：客观描述输入输出、核心约束和边界条件。
- 多解法实现：覆盖暴力解法、最优解法和拓展变形解法，并保留完整 Python 代码。
- 复杂度对比：横向比较三种方案的时间复杂度、空间复杂度、优缺点和适用场景。
- 工程选型思考：说明不同数据规模、线上服务、高并发场景下的算法选择。
- 高频面试问答：整理可直接口述的追问与标准回答。
- 易错点复盘：记录编码和面试表达中容易踩坑的边界。

## 覆盖考点

当前文档覆盖 LeetCode Hot100 中的高频算法模块，包括但不限于：

- 哈希表与集合
- 双指针与滑动窗口
- 字符串处理与子串问题
- 数组、矩阵与前后缀技巧
- 链表与二叉树
- 图论、回溯与搜索
- 二分查找、栈、堆和贪心
- 动态规划与多维动态规划
- 位运算、投票算法和原地标记技巧

## 补充说明

所有代码均以 Python 为主，可直接用于本地复盘和面试手写练习。文档重点在于强化面试口述逻辑、复杂度分析、工程落地取舍和同类题迁移能力。仓库将持续更新完善。
